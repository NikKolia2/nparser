Запуск ./docker-up
Остановка ./docker-down
Подключится к контейнеру с php
Создать новый проект
./agelar create project project_name
project_name должен быть валидным, чтобы на его основании могла создаться директория и база данных (без пробелов и странных символов)
После этой команды создается директория и новая база данных с 4 таблицами.
Для подключения к базе данных через программы
